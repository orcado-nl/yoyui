# Contributing

This project is maintained by **Orcado** and is currently not open to external contributions.

## Who can contribute?

Contributions (pull requests, issues, code reviews) are currently limited to Orcado employees. External pull requests will be closed without review.

Are you an Orcado employee and want to contribute? Follow the steps below.

## Getting started

1. **Clone the repository**
   ```bash
   git clone https://github.com/orcado/<repo-name>.git
   ```

2. **Create a branch**
   ```bash
   git checkout -b feature/short-description
   ```

3. **Commit your changes**
   Use clear commit messages that explain what changed and why.

4. **Push and open a pull request**
   ```bash
   git push origin feature/short-description
   ```
   Open a pull request against the `main` branch and request review from at least one colleague.

## Code style

- Follow the existing code conventions in this project.
- Make sure existing tests pass, and add new tests where relevant.

## Questions

For questions, reach out via Orcado's internal channels.
